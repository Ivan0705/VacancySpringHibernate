# README #

1. Напишите приложение, собирающее вакансии с портала HH.RU в г. Новосибирск, проф. Область «Информационные технологии, интернет, телеком», сохраняющий их в базу данных и предоставляющий возможности по просмотру и поиску.
2. Данные нужно принимать, используя API портала. Документация на API находится в общем доступе http://api.hh.ru
3. Допускается сохранять и отображать не все данные о вакансиях, а только основные: название, дата публикации, организация, зарплата.
4. Возможности для постраничного вывода списка вакансий и возможность сортировки.
5. возможность настройки параметров экспорта с портала HH.RU (регион)
