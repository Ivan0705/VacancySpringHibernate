# README #

1. Напишите приложение, собирающее вакансии с портала HH.RU проф. область: «Информационные технологии, интернет, телеком», сохраняющий их в базе данных и предоставляющий возможности по просмотру и поиску.
2. Данные нужно принимать, используя API портала. Документация на API находится в общем доступе http://api.hh.ru
3. Допускается сохранять и отображать не все данные о вакансиях, а только основные: название, дата публикации, организация, зарплата.
4. Необходимо создать постраничный вывод списка вакансий.
5. Ввести величину топ-зарплаты.
6. Использовать возможность выбора городов СНГ
